# Overview

## I. How to start

### Environment Requirement

```
nodejs version >= 12.16.3
npm version >= 6.14.4
yarn version >= 1.22.4
```

### Start the app

#### 1. Make sure to install all packages befort start

```
yarn install
```

#### 2. Start the development environment

```
yarn start
```

#### 3. Create production build

```sh
yarn build
```
